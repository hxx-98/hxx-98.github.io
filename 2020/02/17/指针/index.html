  • 记忆技巧

    (1) const = 常量，* = 指针。

    (2) const 紧挨谁，谁不能修改。

    如：const int* p ：常量指针，const 紧挨 int*，所以 *p 不可以改。

    ​ int* const p ：指针常量，const紧挨 p，所以 p 不可以修改。
