MySQL安装

  • 参考链接

MySQL修改用户密码

  • ALTER USER ‘用户名‘@’localhost’ IDENTIFIED BY ‘密码’

MySQL启动

  • 命令行

    启动：net start mysql

    关闭：net stop mysql

MySQL登录

  • 登录

    • 本地登录：mysql -u[用户名] -p[密码]
    • IP登录：mysql -h[IP地址] -u[用户名] -p[密码]

  • 退出：exit 或者 quit
