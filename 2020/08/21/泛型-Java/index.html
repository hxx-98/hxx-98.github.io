泛型-Java

定义含有泛型的类

定义含有泛型的方法

  • 修饰符 <泛型> 返回值类型 方法名(参数列表){ ………. }

含有泛型的接口

  • 修饰符 interface 接口名 <代表泛型的变量> { }

泛型通配符<?>

  • 代表任意类型，不能定义时使用

  • 泛型的上限：只接受该类型及其子类

    类型名 <? extends 类 > 对象名称

  • 泛型的下限：只接受该类型极其父类

    类型名 <? super 类 > 对象名称
