Map集合-Java

Map集合概述

  1. 双列集合每一个元素包括两个值key和value，两者数据类型可以相同，也可以不用
  2. key不可以重复，value可以重复
  3. key和value一一对应

Map常用子类

  • HashMap:无序集合，底层是哈希表。当存储自定义类型键值，需要重写toString和equals方法。
  • LinkedHashMap：HashMap的子类，有序集合，存储和取出顺序相同。
